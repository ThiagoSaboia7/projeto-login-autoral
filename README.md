<h2>📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de uma Tela de Login minimalista, moderna e elegante, construída inteiramente do zero. O foco principal foi criar uma interface de usuário (UI) atraente através do uso inteligente de gradientes de cores e alinhamento perfeito utilizando CSS Flexbox.</h2>
<br></br>

<h2>✨ Funcionalidades em Destaque</h2>

<ul>
<h3><li>🎯 Alinhamento Centralizado: O uso de display: flex, align-items: center e justify-content: center garante que o formulário fique perfeitamente no meio da tela em qualquer monitor.</li> <br>

<li>🎨 Design com Gradiente: O fundo da página possui um linear-gradient suave, transitando do cinza escuro para um azul moderno.</li> <br>

<li>📱 Estrutura Responsiva base: Utilização da tag <meta name="viewport"> e larguras relativas/fixas que evitam a quebra do layout.</li> <br>
</h3></ul>
